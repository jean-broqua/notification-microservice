# Notification Microservice

Complete microservice to handle notifications.

## Used:

- NestJs
- Prisma
- SQLite
- Kafka

## Setup:

Start:

```
yarn install && yarn start
```

Test:

```
yarn test
```

## Applied Concepts:

- SOLID Principles
- Dependency Injection
- Dependency Inversion
- Repository Pattern
- Unit Tests
- In-Memory Database
- Factory Pattern
- REST
